# 🧮 Formulas and Functions

## Median Salary Calculation

The following formula calculates the median salary based on multiple criteria such as job title, country, and job schedule type.

=MEDIAN(
IF(
(jobs[job_title_short]=A2)*
(jobs[job_country]=country)*
(ISNUMBER(SEARCH(type,jobs[job_schedule_type])))*
(jobs[salary_year_avg]<>0),
jobs[salary_year_avg]
)
)

Key Logic

Multi-Criteria Filtering  
The formula filters the dataset by job title, country, job schedule type, and excludes missing salary values.

Array Formula  
The MEDIAN() function is applied over an array generated by the nested IF() conditions.

Purpose  
This formula powers the dashboard tables and returns the median salary for selected job titles, countries, and schedule types.

# Job Schedule Type Filtering

A formula was used to generate a clean list of job schedule types:

=FILTER(J2#,(NOT(ISNUMBER(SEARCH("and",J2#))+ISNUMBER(SEARCH(",",J2#))))*(J2#<>0))

Formula Explanation

Unique List Generation  
This formula filters out entries containing "and", removes entries containing commas, and excludes zero values.

Purpose  
The resulting list provides clean job schedule categories that are used as filters in the dashboard.
